# 第二章：量化参与方式全景图

普通投资者想参与量化交易,有多种方式可选。本章将详细介绍四种主要参与方式,帮助你找到最适合自己的路径。

---

## 2.1 四种主要参与方式概览

### 全景图

```
                                           量化交易参与方式
                                                   │
                      ┌────────────────┬───────────┴────────┬──────────────────┬
                      │                │                    │                  │
                      │                │                    │                  │
               ┌──────▼──────┐  ┌──────▼──────┐      ┌──────▼──────┐    ┌──────▼──────┐
               │ 买量化基金   │  │ 量化跟单     │      │ 使用工具     │   │  自主开发    │
               │  (最简单)    │  │ (跟高手)    │      │(无需编程)    │    │  (需编程)   │
               └─────────────┘  └─────────────┘      └─────────────┘    └─────────────┘
                     │                  │                  │                  │
                 ⭐门槛最低        ⭐⭐选择丰富       ⭐⭐⭐快速验证   ⭐⭐⭐⭐⭐完全自主
                   100元起             几万元起             几万元起           几万元起
                  年化8-20%          年化10-30%          年化15-30%           年化20%+
```

|---------|------|---------|---------|---------|---------|

```

愿意学习,不想编程       →    使用量化工具
技术背景/深度学习兴趣   →    自主开发策略
```

## 2.2 购买量化基金(最简单)

### 什么是量化基金

量化基金就是用量化策略管理的基金,你把钱交给专业的基金经理,他们用量化模型投资,你坐享收益。

**就像**:
- 你不会做菜,去餐厅吃饭
- 你不懂量化,买量化基金

### 两大类型

#### 1. 公募量化基金

**特点**:
- 监管严格,相对安全
- 门槛低,100元起
- 流动性好,T+1赎回
- 收费透明

**主要品种**:

**a) 指数增强基金**(最常见)
```
运作方式:
- 跟踪某个指数(如沪深300)
- 用量化模型选股,超越指数
- 例如: 90%仓位跟踪指数 + 10%量化增强

收益目标:
- 指数收益 + 超额收益
- 超额收益通常: 年化5-15%

代表产品:
- 富国沪深300指数增强
- 景顺长城沪深300增强
- 华泰柏瑞沪深300ETF联接
```

**b) 主动量化基金**
```
运作方式:
- 不跟踪指数,纯量化选股
- 基金经理有更大自由度
- 仓位、行业可灵活调整

收益目标:
- 绝对收益
- 年化15-25%

代表产品:
- 华泰柏瑞量化优选
- 长信量化先锋
- 光大保德信量化核心
```

**c) 量化对冲基金**
```
运作方式:
- 股票多头 + 股指期货空头
- 对冲市场风险,追求绝对收益
- 市场涨跌对净值影响小

收益目标:
- 年化8-12%
- 波动小,回撤小

代表产品:
- 海富通阿尔法对冲
- 华泰柏瑞量化对冲
- 富国绝对收益
```

**如何购买**:
```
渠道:
1. 基金公司官网
2. 支付宝/微信理财通
3. 天天基金/蛋卷基金
4. 银行/券商APP

步骤:
1. 搜索基金名称/代码
2. 了解基金详情
3. 点击买入
4. 输入金额
5. 确认支付
```

#### 2. 私募量化基金

**特点**:
- 门槛高,100万起
- 策略更灵活
- 不公开宣传
- 业绩分成(一般20%)

**主要策略**:

**a) 股票多头**
```
- 纯多头量化选股
- 类似公募主动量化
- 但仓位可到0-100%灵活
- 年化目标: 20-30%
```

**b) 指数增强**
```
- 与公募类似
- 但可用衍生品增强收益
- 规模大,最主流
- 年化目标: 指数+10-20%
```

**c) 市场中性**
```
- 股票多头 + 期货空头对冲
- 追求与市场无关的收益
- 回撤小,夏普比率高
- 年化目标: 10-18%
```

**d) CTA策略**
```
- 商品期货趋势跟踪
- 与股票相关性低
- 波动大,适合配置
- 年化目标: 15-30%
```

**如何购买**:
```
前提: 合格投资者
- 金融资产300万以上,或
- 近3年年均收入50万以上

渠道:
1. 私募直销
2. 第三方财富管理公司
3. 私人银行
4. 券商资管

步骤:
1. 联系销售机构
2. 填写合格投资者问卷
3. 了解产品详情
4. 签署合同
5. 打款认购
```

### 优缺点

#### 优点

**1. 省心省力**
```
不需要:
- 学习编程
- 研究策略
- 每天盯盘
- 自己交易

只需要:
- 选择基金
- 定期查看净值
- 必要时调整
```

**2. 专业管理**
```
基金团队:
- 专业基金经理
- 量化研究员
- 交易员
- 风控团队

资源优势:
- 专业数据源
- 先进模型
- 低交易成本
- 机构研究支持
```

**3. 风险分散**
```
一只基金通常持有:
- 几十到上百只股票
- 分散行业
- 分散风格
- 降低单一股票风险
```

**4. 监管保护**
```
公募基金:
- 证监会监管
- 托管银行监管
- 信息披露透明
- 资金安全有保障
```

#### 缺点

**1. 管理费用**
```
公募基金费用:
- 管理费: 0.6-1.5%/年
- 托管费: 0.1-0.25%/年
- 申购费: 0-1.5%(可打折)
- 赎回费: 0-0.5%(持有时间越长越低)

私募基金费用:
- 管理费: 1-2%/年
- 业绩报酬: 20%(超额收益)
- 例如: 收益20%,你拿16%,管理人拿4%
```

**2. 收益天花板**
```
公募基金:
- 受规模限制
- 规模大了策略容量有限
- 很难有特别高的超额收益

私募基金:
- 业绩好了规模扩大
- 规模扩大后收益下降
- 马太效应
```

**3. 信息不对称**
```
投资者不知道:
- 具体持仓(公募每季度公布,滞后)
- 交易逻辑
- 策略细节
- 只能看净值结果
```

**4. 被动接受**
```
无法控制:
- 基金经理的操作
- 策略的调整
- 风险敞口
- 只能选择买入或赎回
```

### 如何选择量化基金

详见第三章,这里简单介绍:

**关键指标**:
```
1. 历史业绩
   - 看3年以上业绩
   - 注意不同市场环境表现

2. 风险指标
   - 最大回撤
   - 波动率
   - 夏普比率

3. 超额收益稳定性
   - 指数增强要看超额收益
   - 是否持续稳定

4. 基金经理
   - 从业经验
   - 过往业绩
   - 团队稳定性

5. 规模
   - 不要太小(流动性问题)
   - 不要太大(策略容量)
   - 5-50亿较合适
```

## 2.3 量化跟单(跟随高手)

### 什么是量化跟单

**简单理解**:
- 别人开发了一个量化策略
- 在平台上分享
- 你选择跟随这个策略
- 策略发出信号,你的账户自动同步交易

**类比**:
```
就像:
- 别人是大厨,有菜谱
- 你在家按菜谱做菜
- 但是自动化的:别人炒菜,你家锅也自动炒
```

### 主要平台

#### 1. 券商跟单平台

**东方财富/同花顺**:
```
优势:
- 券商背书,相对可靠
- 策略经过筛选
- 资金在自己账户,安全

如何使用:
1. 开通交易权限
2. 浏览策略市场
3. 查看策略历史业绩
4. 选择策略跟单
5. 设置跟单金额
6. 自动同步交易
```

**代表平台**:
- 东方财富: "东方财富金工"
- 同花顺: "问财智投"
- 平安证券: "AI慧炒股"

#### 2. 第三方跟单平台

**聚宽/优矿等**:
```
特点:
- 策略更丰富
- 个人也能发布策略
- 需要连接券商账户

风险:
- 策略质量参差不齐
- 需要仔细筛选
```

#### 3. 社交交易平台

**雪球/eToro等**:
```
特点:
- 跟单真实投资者
- 不一定是量化策略
- 社交属性强

适合:
- 想跟随牛人
- 希望能交流学习
```

### 跟单流程

#### Step 1: 选择平台
```
考虑因素:
- 平台可靠性
- 策略丰富度
- 费用成本
- 用户评价
```

#### Step 2: 筛选策略
```
筛选条件:
1. 运行时长(至少1年)
2. 累计收益
3. 最大回撤
4. 夏普比率
5. 胜率
6. 跟单人数

重点看:
- 策略稳定性
- 回撤控制
- 不要只看收益
```

#### Step 3: 深入分析
```
查看:
1. 历史净值曲线
2. 月度收益分布
3. 持仓分析
4. 换手率
5. 策略说明

警惕:
- 短期暴涨的策略
- 历史太短的策略
- 回撤特别大的
- 说明含糊的
```

#### Step 4: 小资金测试
```
建议:
1. 先用小资金(1-5万)
2. 观察1-3个月
3. 看实际表现与回测是否一致
4. 满意再加大资金
```

#### Step 5: 持续监控
```
定期检查:
- 净值变化
- 回撤情况
- 与预期是否一致

出现以下情况考虑停止:
- 连续大幅回撤
- 策略逻辑失效
- 与历史表现差异大
```

### 跟单模式

#### 1. 完全跟随
```
设置:
- 按策略100%同步

例如:
- 策略买入某股5%仓位
- 你也买入5%仓位

适合:
- 完全信任策略
- 资金量不大
```

#### 2. 比例跟随
```
设置:
- 按一定比例跟随
- 例如: 策略仓位50%

例如:
- 策略买入某股10%仓位
- 你实际买入5%仓位

适合:
- 想控制风险
- 多策略组合
```

#### 3. 固定金额跟随
```
设置:
- 固定投入金额

例如:
- 无论策略仓位
- 你固定投入10万

适合:
- 试探性跟单
- 组合多个策略
```

### 优缺点

#### 优点

**1. 门槛低**
```
不需要:
- 编程能力
- 深入研究
- 开发策略

只需要:
- 会选择策略
- 设置跟单
```

**2. 选择丰富**
```
可以:
- 浏览大量策略
- 对比选择
- 多策略组合
- 随时切换
```

**3. 透明度高**
```
能看到:
- 历史净值
- 持仓情况
- 交易记录
- 策略说明

比基金更透明
```

**4. 资金可控**
```
优势:
- 资金在自己账户
- 随时可以停止
- 可以部分跟单
- 灵活性高
```

#### 缺点

**1. 策略质量难辨**
```
风险:
- 历史业绩可能过拟合
- 策略可能失效
- 虚假宣传
- 数据造假

需要:
- 仔细筛选
- 小资金测试
- 持续监控
```

**2. 滑点成本**
```
问题:
- 信号有延迟
- 你的成交价可能不如策略
- 尤其是快速交易的策略

例子:
- 策略10.00元买入
- 你收到信号时已经10.05
- 每次差5分,累积下来不少
```

**3. 跟单拥挤**
```
问题:
- 好的策略很多人跟
- 同时买卖
- 冲击成本大
- 策略失效快

例子:
- 策略买入某小盘股
- 1000人同时跟单
- 股价被拉高
- 后面跟的人成本高
```

**4. 费用成本**
```
费用:
- 平台服务费
- 跟单分成(10-30%)
- 自己的交易佣金

例如:
- 策略收益20%
- 扣除20%分成
- 你实际16%
- 再扣佣金和费用
```

### 成功跟单要点

#### 1. 选择策略
```
✓ 运行时间足够长(1年以上)
✓ 净值曲线平滑
✓ 回撤可控(<20%)
✓ 收益稳定(不要追求暴利)
✓ 策略逻辑清晰

✗ 短期暴涨
✗ 巨大回撤后反弹
✗ 换手率极高
✗ 说明含糊不清
```

#### 2. 资金管理
```
分散跟单:
- 不要all in一个策略
- 3-5个策略组合
- 不同风格分散

逐步建仓:
- 先小资金测试
- 观察1-3个月
- 逐步加大
```

#### 3. 风险控制
```
设置止损:
- 单个策略最大亏损
- 整体回撤限制

定期评估:
- 每月检查
- 与预期对比
- 及时调整
```

## 2.4 使用量化工具(无需编程)

### 什么是量化工具

**定义**:
- 可视化搭建量化策略的工具
- 不需要写代码
- 通过拖拽、选择等方式构建策略

**类比**:
```
就像:
- 编程是手写代码盖房子
- 量化工具是用乐高积木搭建
- 效果类似,但降低了技术门槛
```

### 主要工具类型

#### 1. 可视化策略平台

**聚宽可视化策略**:
```
特点:
- 拖拽式构建策略
- 选择指标和条件
- 自动生成代码
- 云端回测

适合:
- 有交易思路
- 不会编程
- 想快速验证想法
```

**优矿策略工坊**:
```
类似聚宽
- 界面更简洁
- 模板丰富
- 适合新手
```

**米筐策略编辑器**:
```
- 积木式搭建
- 丰富的策略模块
- 社区策略分享
```

#### 2. 券商量化工具

**同花顺条件单**:
```
功能:
- 设置触发条件
- 自动下单

例如:
- 股价突破20日均线 → 自动买入
- 盈利10% → 自动卖出
- 亏损5% → 自动止损

适合:
- 简单的条件触发
- 懒人交易
```

**同花顺智能盯盘**:
```
- 设置监控条件
- 达到条件发送提醒
- 也可自动交易
```

**通达信选股器**:
```
- 技术指标选股
- 公式系统
- 不需要编程(使用内置公式)
- 也可以自定义公式

适合:
- 选股需求
- 技术分析爱好者
```

**东方财富AI选股**:
```
- 多维度选股
- 行业、财务、技术指标
- 一键筛选

适合:
- 快速选股
- 基本面+技术面结合
```

#### 3. 第三方量化工具

**量化交易终端**:
```
- 集成行情、交易、策略
- 可视化操作
- 本地运行

代表:
- 文华财经
- 快期
- 易盛
```

**策略工厂平台**:
```
- 提供策略模板
- 修改参数即可
- 一键回测和实盘
```

### 使用流程

以聚宽可视化策略为例:

#### Step 1: 注册登录
```
访问: joinquant.com
注册账号
登录平台
```

#### Step 2: 创建策略
```
1. 点击"创建策略"
2. 选择"可视化策略"
3. 命名策略
```

#### Step 3: 搭建策略逻辑
```
拖拽模块:
┌─────────────┐
│ 选股模块    │  ← 选择股票池
└─────────────┘
       ↓
┌─────────────┐
│ 过滤条件    │  ← 设置筛选条件
└─────────────┘
       ↓
┌─────────────┐
│ 买入信号    │  ← 设置买入条件
└─────────────┘
       ↓
┌─────────────┐
│ 卖出信号    │  ← 设置卖出条件
└─────────────┘
       ↓
┌─────────────┐
│ 风控设置    │  ← 止损止盈
└─────────────┘
```

#### Step 4: 参数设置
```
例如: 双均线策略
- 短期均线: 5日
- 长期均线: 20日
- 买入: 短期均线上穿长期
- 卖出: 短期均线下穿长期
```

#### Step 5: 回测
```
设置:
- 回测时间: 2020-01-01 到 2023-12-31
- 初始资金: 100万
- 基准: 沪深300

运行回测:
```

### 简单示例

#### 策略: 双均线

**策略逻辑**:
```
买入信号: 5日均线上穿20日均线
卖出信号: 5日均线下穿20日均线
```

**完整代码**:
```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import tushare as ts

# 1. 获取数据
def get_data(code, start_date, end_date):
    """获取股票数据"""
    df = ts.get_k_data(code, start=start_date, end=end_date)
    df['date'] = pd.to_datetime(df['date'])
    df.set_index('date', inplace=True)
    return df

# 2. 计算指标
def calculate_signals(df):
    """计算双均线策略信号"""
    # 计算均线
    df['ma5'] = df['close'].rolling(window=5).mean()
    df['ma20'] = df['close'].rolling(window=20).mean()

    # 生成信号
    df['signal'] = 0
    df['signal'] = np.where(df['ma5'] > df['ma20'], 1, 0)

    # 计算仓位变化(买入/卖出点)
    df['position'] = df['signal'].diff()

    return df

# 3. 回测
def backtest(df, initial_capital=100000):
    """简单回测"""
    # 初始化
    capital = initial_capital
    position = 0  # 持仓数量
    trades = []

    for date, row in df.iterrows():
        # 买入信号
        if row['position'] == 1:
            position = capital / row['close']  # 全仓买入
            trades.append({
                'date': date,
                'action': 'buy',
                'price': row['close'],
                'position': position
            })
            print(f"{date.date()}: 买入 {position:.0f}股 @{row['close']:.2f}")

        # 卖出信号
        elif row['position'] == -1 and position > 0:
            capital = position * row['close']  # 全仓卖出
            trades.append({
                'date': date,
                'action': 'sell',
                'price': row['close'],
                'capital': capital
            })
            print(f"{date.date()}: 卖出 {position:.0f}股 @{row['close']:.2f}, 资金:{capital:.0f}")
            position = 0

    # 最终市值
    final_value = capital if position == 0 else position * df.iloc[-1]['close']
    total_return = (final_value - initial_capital) / initial_capital * 100

    print(f"\n初始资金: {initial_capital:.0f}")
    print(f"最终市值: {final_value:.0f}")
    print(f"总收益率: {total_return:.2f}%")

    return trades, final_value

# 4. 可视化
def plot_strategy(df):
    """绘制策略图"""
    plt.figure(figsize=(14, 7))

    # 价格和均线
    plt.plot(df.index, df['close'], label='价格', alpha=0.5)
    plt.plot(df.index, df['ma5'], label='MA5')
    plt.plot(df.index, df['ma20'], label='MA20')

    # 买入点
    buy_signals = df[df['position'] == 1]
    plt.scatter(buy_signals.index, buy_signals['close'],
                color='red', label='买入', marker='^', s=100)

    # 卖出点
    sell_signals = df[df['position'] == -1]
    plt.scatter(sell_signals.index, sell_signals['close'],
                color='green', label='卖出', marker='v', s=100)

    plt.title('双均线策略')
    plt.xlabel('日期')
    plt.ylabel('价格')
    plt.legend()
    plt.grid(True)
    plt.show()

# 5. 主程序
if __name__ == '__main__':
    # 参数设置
    code = '600519'  # 贵州茅台
    start_date = '2020-01-01'
    end_date = '2023-12-31'

    # 获取数据
    print("获取数据...")
    df = get_data(code, start_date, end_date)

    # 计算信号
    print("计算信号...")
    df = calculate_signals(df)

    # 回测
    print("\n开始回测...")
    trades, final_value = backtest(df)

    # 可视化
    print("\n绘制图表...")
    plot_strategy(df)
```

```
1. 先买基金,边投边学

4. 逐步过渡到自主开发

直接选择跟单或基金
不是所有人都要学编程
```

#### 4. 目标维度

```

```

**目标: 职业发展**:
```
必选: 自主开发

- 建立专业能力
- 可转行到量化私募
```

### 常见误区

```
错误想法:

- 容易挫败放弃

建议:
- 先用简单方式参与
- 边投资边学习
- 循序渐进
```

#### 误区2: 只看收益率
```
错误想法:
"自主开发收益最高,我选这个!"

问题:
- 忽略了时间成本
- 忽略了学习成本
- 忽略了失败风险

建议:
- 综合考虑时间、能力、目标
- 不要只看收益
```

#### 误区3: 贪多求全
```
错误想法:
```

## 本章小结

本章介绍了量化交易的四种参与方式:

### 四种方式总结

1. **购买量化基金**:最简单,适合所有人
   - 公募: 100元起,监管严,透明度高
   - 私募: 100万起,策略灵活,收益潜力大

2. **量化跟单**:跟随高手,省心省力
   - 门槛适中,选择丰富
   - 关键是筛选优质策略

3. **使用工具**:无需编程,降低门槛
   - 可视化搭建策略
   - 快速验证想法

4. **自主开发**:完全掌控,潜力最大
   - 学习成本高,时间投入大
   - 但收益潜力和学习价值最大
