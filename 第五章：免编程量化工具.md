## 5.2 券商量化工具

### 同花顺条件单/智能盯盘

#### 条件单功能

**什么是条件单**:
```
定义:
设置触发条件,条件满足自动下单

例如:
- 股价跌到20元,自动买入
- 持仓盈利10%,自动卖出
- 跌破止损价,自动止损

优势:
- 不需要盯盘
- 24小时监控
- 纪律执行
```

**开通方式**:
```
路径:
同花顺APP → 交易 → 条件单

条件:
- 可能需要一定资产
- 签署协议
- 部分券商支持

支持券商:
- 查看同花顺支持列表
- 不是所有券商都支持
```

#### 条件单类型

**1. 价格触发**:
```
买入条件:
- 价格 ≤ 目标价
- 例如: 价格跌到50元买入

卖出条件:
- 价格 ≥ 目标价
- 例如: 价格涨到60元卖出

高级:
- 价格突破某均线
- 价格跌破某均线
```

**2. 涨跌幅触发**:
```
买入:
- 今日涨幅 ≥ 5%
- 追涨策略

卖出:
- 今日跌幅 ≥ -3%
- 止损策略
```

**3. 均线触发**:
```
金叉买入:
- MA5上穿MA20
- 自动买入

死叉卖出:
- MA5下穿MA20
- 自动卖出
```

**4. 成本价触发**:
```
止盈:
- 盈利 ≥ 10%
- 自动卖出

止损:
- 亏损 ≥ -5%
- 自动卖出
```

**5. 时间触发**:
```
定时:
- 每周一开盘买入
- 每月最后一天卖出

持有时间:
- 持有超过30天自动卖出
```

#### 公式系统

**什么是公式**:
```
定义:
用公式语言编写指标和选股条件

类似编程但更简单:
- 语法简单
- 专门为股票设计
- 不需要专业编程知识

例子:
MA5:=MA(CLOSE,5);    // 5日均线
MA20:=MA(CLOSE,20);  // 20日均线
金叉:=CROSS(MA5,MA20);  // 金叉
卖出:CROSS(MA20,MA5);   // 死叉
```

**编写简单公式**:
```
路径:
通达信 → 功能 → 公式管理器 → 新建

例子1: 双均线选股
MA5:=MA(C,5);
MA20:=MA(C,20);
金叉:CROSS(MA5,MA20);

保存后可以:
- 在选股器中使用
- 在指标窗口显示
```

**常用公式语言**:
```
基本函数:
MA(X,N)      // N日均线
SUM(X,N)     // N日累加
HHV(X,N)     // N日最高
LLV(X,N)     // N日最低
CROSS(A,B)   // A上穿B
COUNT(X,N)   // N日满足X的天数

数据引用:
C / CLOSE    // 收盘价
O / OPEN     // 开盘价
H / HIGH     // 最高价
L / LOW      // 最低价
V / VOL      // 成交量

逻辑:
AND          // 与
OR           // 或
NOT          // 非
>  <  =      // 大于小于等于
```

**实用公式示例**:

**1. 放量突破选股**:
```
MA60:=MA(C,60);          // 60日均线
突破:=C>MA60;            // 收盘价突破均线
放量:=V>MA(V,5)*2;       // 成交量大于5日均量2倍
选出:突破 AND 放量;
```

**2. 低位金叉选股**:
```
MA5:=MA(C,5);
MA20:=MA(C,20);
金叉:=CROSS(MA5,MA20);
低位:=C<HHV(C,60)*0.8;   // 价格处于60日高点80%以下
选出:金叉 AND 低位;
```

**3. 超跌反弹选股**:
```
最高价:=HHV(C,60);
当前价:=C;
跌幅:=(当前价-最高价)/最高价;
超跌:=跌幅<-0.3;         // 从高点跌超30%
反弹:=C>MA(C,5);         // 站上5日均线
选出:超跌 AND 反弹;
```

**错误2: 未来函数**:
```
现象:
回测完美,实盘全错

- 用了未来数据
- 例如用收盘价判断,开盘买

- 仔细检查逻辑

```

回测年化30%

问题:
- 没考虑滑点
- 没考虑冲击成本
- 高换手吃掉收益

解决:
- 设置真实成本
- 佣金万3
- 滑点0.2-0.5%
- 看扣除成本后的收益
```

**错误4: 数据幸存者偏差**:
```
现象:
回测选股完美,都是后来的牛股

问题:
- 自动剔除了退市、ST股票
- 但实盘会遇到

解决:
- 设置包含退市股
- 考虑ST风险
- 更接近真实环境
