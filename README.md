# 人人都能懂的量化交易：从入门到精通


## 🗺️ 快速导航：我应该从哪里开始？

### 💡 我是完全新手
**推荐路径**：认知篇 → 产品篇第三章（量化基金）
- 先了解什么是量化
- 然后选择适合的量化基金产品投资
- **学习时间**：2-3小时

### 🔄 我想跟单
**推荐路径**：认知篇 → 产品篇第四章（跟单平台） → 知识篇第十章（策略评估）
- 了解量化基础知识
- 选择跟单平台和策略
- 学会评估策略质量
- **学习时间**：4-5小时

### 🎮 我想用工具但不会编程
**推荐路径**：认知篇 → 产品篇第五章（免编程工具） → 知识篇
- 了解量化基础
- 学习使用可视化工具
- 理解策略和风险管理
- **学习时间**：1-2天

### 💻 我想自己开发（有编程基础）
**推荐路径**：认知篇 → 知识篇 → 技术篇 → 实战篇
- 完整学习路径
- 从基础到进阶
- **学习周期**：2-3个月

### 💻 我想自己开发（零编程基础）
**推荐路径**：认知篇 → 知识篇 → 技术篇第十三章（编程入门） → 第十四、十五章 → 持续学习
- 先学Python基础
- 再学量化应用
- **学习周期**：3-4个月

### 🔬 我是专业开发者
**推荐路径**：快速浏览认知篇、知识篇 → 重点学习进阶篇 → 实战篇
- 重点关注框架、工具、实战经验
- 深入研究因子、ML、高级策略
- **学习周期**：1-2个月
