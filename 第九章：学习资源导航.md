### 重要论文

#### 经典必读

**1. "The Cross-Section of Expected Stock Returns" (Fama & French, 1992)**:
```
贡献: 提出三因子模型
影响: 奠定因子投资基础

核心:
- 市场因子(Beta)
- 规模因子(SMB)
- 价值因子(HML)
```

**2. "Returns to Buying Winners and Selling Losers" (Jegadeesh & Titman, 1993)**:
```
贡献: 动量效应
影响: 动量策略理论基础

发现: 过去表现好的股票未来继续好
```

**3. "...and the Cross-Section of Expected Returns" (Carhart, 1997)**:
```
贡献: 四因子模型
影响: 加入动量因子

扩展: Fama-French + 动量
```

#### 近期前沿

**1. 机器学习在资产定价中的应用**:
```
作者: Gu, Kelly, Xiu (2020)
期刊: Review of Financial Studies

内容:
- 深度学习预测收益
- 非线性因子模型
- 大数据方法
```

**2. 另类数据与投资**:
```
领域: 卫星图像,信用卡数据

应用:
- 预测零售销售
- 监测经济活动
- 情绪分析

## 8.6 实战提升路径

### 新手(0-6个月)

**阶段目标**: 建立基础,理解概念

**学习路径**:
```
1. 阅读
   - 《打开量化投资的黑箱》
   - 《聪明的投资者》

2. 编程
   - Python基础(廖雪峰教程)
   - pandas数据处理

3. 实践
   - 注册聚宽账号
   - 学习聚宽课堂
   - 编写简单双均线策略
   - 回测验证

4. 数据
   - 注册Tushare Pro
   - 学习数据获取
   - 本地数据处理
```

**检验标准**:
- ✓ 能独立获取数据
- ✓ 能编写简单策略
- ✓ 理解回测基本概念
- ✓ 能计算收益率、夏普比率

### 初级(6个月-1年)

**阶段目标**: 深入策略,开始实盘

**学习路径**:
```
1. 阅读
   - 《量化交易：如何建立自己的算法交易》
   - 《Python金融大数据分析》

2. 策略
   - 均值回归策略
   - 动量策略
   - 多因子选股
   - 回测优化

3. 实盘
   - 开通QMT
   - 小资金实盘(1-5万)
   - 跟踪实盘表现

4. 数据
   - 升级9db入门版
   - 使用专业数据
```

**检验标准**:
- ✓ 熟练使用回测框架
- ✓ 能开发3种以上策略
- ✓ 实盘运行无技术问题
- ✓ 理解风险指标

### 中级(1-2年)

**阶段目标**: 策略进阶,稳定盈利

**学习路径**:
```
1. 阅读
   - 《主动投资组合管理》
   - 《金融时间序列分析》
   - 学术论文(每月2-3篇)

2. 策略
   - 因子挖掘
   - 组合优化
   - 风险管理
   - 机器学习应用

3. 系统
   - 搭建完整交易系统
   - 数据库管理
   - 监控告警
   - 自动化执行

4. 实盘
   - 资金规模扩大
   - 多策略运行
   - 业绩归因分析
```

**检验标准**:
- ✓ 实盘稳定盈利
- ✓ 年化收益>15%
- ✓ 最大回撤<10%
- ✓ 有完整交易系统

### 高级(2年+)

**阶段目标**: 专业化,规模化

**学习路径**:
```
1. 深度研究
